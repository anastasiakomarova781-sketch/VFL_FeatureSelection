## О проекте

Проект реализует серверную логику процессов вертикального федеративного обучения.
Создано на основе [UCBFL framework](https://git.ucb.local/projects/CC/repos/ucbfl/)
Компоненты  выдаются в виде приложений на базе python упакованных в докер образ. На данный момент имеется реализация следующих приложений:
- Сервер обучения для активной стороны
- Сервер обучения для пассивной стороны
- Сервер предсказания для активной стороны
- Сервер предсказания для пассивной стороны


## Структура проекта
- /contracts - proto api
- /doc - документация
- /docker - файлы необходимые для сборки докер образа
- /example - примеры использования в виде клиента на питоне. Тестовые данные. 
- /external - внешние модули. Например  [UCBFL framework](https://git.ucb.local/projects/CC/repos/ucbfl/)
- /python - исходный код на python


## Сборка докер образа

```
docker build --progress=plain --no-cache -f docker/runtime.docker -t ucbvfl-server .
```
или
```
./docker_build.sh
```

## Запуск серверов из образа
См примеры:
```
./docker_run.sh
```

## Генерация кода для клиента из proto файлов

```
./generate_proto.sh
```
